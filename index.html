Welcome to SportDataLibrary pages!

If you're a European Soccer Lover you'll love this library

introduction

This Library was brought to you as part of the course “Programming languages” as a final project.

We chose this subject because we both love European soccer (rival clubs) and we wanted to supply an easy way to get information about main European leagues, clubs and players.

The Web Service

We want to give credit to the web service we derived the data from: http://api.football-data.org

We also want to thank the owner of the web service, Daniel Freitag, for very organized API.

Manual

In order to use “SportDataLibrary” you’ll have to start by creating an instance of the interface “ISoccerDataService” and instantiate it.

you can do so by using the static method “GetSoccerDataService” of the class “SoccerDataFactory” and choose the web service you prefer (only one web service for now).

That’s it! You are now using our library!

Watch the documentation for more information about the services we supply.
